# Project: Flight Price AI Agent

## Goal
24/7 personal agent that monitors flight prices and alerts via Telegram
ONLY when a price is at its historical cheapest AND ML forecasts prices will rise.

## Home Airport
NAG (Dr. Babasaheb Ambedkar International Airport, Nagpur, India)

## Core Rule
Never alert based on just "it's the lowest today" — must compare against
full historical record AND confirm ML model says price will not drop further.

## Owner
Ishu Jaiswal · Personal use first · Nagpur, Maharashtra

## Constraints
- Total budget: ₹0–₹150/month (Oracle VPS is free)
- No external dependencies beyond listed tech stack
- All data stored locally (privacy)
- System must survive VPS restart without data loss
